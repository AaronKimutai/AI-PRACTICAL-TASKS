## Tasks Overview

### Task One: MNIST Dataset Classification
* **Objective:** Download the MNIST dataset and implement a program to distinguish digits 0-9.
* **Implementation:** * Utilizes `TensorFlow` and `Keras` to load the dataset.
    * Implements a Neural Network to classify handwritten digits with high accuracy.
    * Normalizes data for optimized training performance.

### Task Two: Constraint Satisfaction Program (CSP)
* **Objective:** Solve map coloring problems where no two adjacent regions share the same color.
* **Sub-Tasks:**
    1. **Map of Australia:** Coloring five regions using three colors (Red, Green, Blue).
    2. **Nairobi Sub-counties:** Coloring the 17 sub-counties of Nairobi using the least possible number of colors.
* **Visualization:** Uses `networkx` and `matplotlib` to generate a graph-based visual representation of the colored map to verify constraints.
