**Objective:**  
Solve map coloring problems where no two adjacent regions share the same color.

**Sub-Tasks:**
- **Map of Australia:** Coloring five regions using three colors (Red, Green, Blue).
- **Nairobi Sub-counties:** Coloring the 17 sub-counties of Nairobi using the least possible number of colors.

**Visualization:**
- Uses `networkx` and `matplotlib` to generate a graph-based visual representation of the colored map to verify constraints.

## Task Four: Search Algorithms (BFS and DFS)
**Objective:**  
Implement Breadth First Search (BFS) and Depth First Search (DFS) algorithms to explore a graph and find a path from an initial node to a goal node.

**Implementation:**
- Represented the problem using a graph (adjacency list).
- Implemented **Breadth First Search (BFS)** to explore nodes level by level and ensure shortest path discovery.
- Implemented **Depth First Search (DFS)** to explore paths deeply before backtracking.
- Both algorithms output the search path from the start node to the goal node.
- Used well-structured Python implementations with clear annotations for readability and understanding.
